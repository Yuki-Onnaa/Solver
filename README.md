# Quiz Solver

## Description
Quiz Solver is a tool designed to help users solve quizzes with ease. It offers several features to streamline the quiz-taking experience.

## Installation Instructions
To install Quiz Solver, clone the repository:
```bash
git clone https://github.com/Yuki-Onnaa/Solver.git
cd Solver
```

Then, install the required dependencies:
```bash
npm install
```

## Setup Instructions for the .env File
1. Copy the provided `.env.example` file to `.env`:
   ```bash
   cp .env.example .env
   ```
2. Update the `.env` file with your configuration settings.

## Usage Instructions
### Manual
To use the manual feature, simply start the application and select the manual option from the menu. You can enter the quiz questions one by one for assistance.

### Auto-Scan
To use the auto-scan feature, place your quiz in front of your webcam and select the auto-scan option. The software will automatically capture and interpret the questions for you.

### Screenshot
You can also upload a screenshot of your quiz. Select the screenshot option and choose the image file containing the quiz questions for processing.
